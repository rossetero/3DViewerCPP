3DViewer

О программе

Программа для просмотра каркасных 3D моделей

Инструкция

Откройте .obj файл чтобы просмотреть его в программе, повращать и подвигать его в разные стороны. Программа поддерживает изменение цвета фона и модели, а также сохраняет эти параметры.

Установка

Для того, чтобы установить программу, вам нужно использовать команду make install.

Запуск

Для того, чтобы начать запустить программу, вам нужно использовать команду make run.

Удаление

Для удаления программы вы можете использовать команду make uninstall.
